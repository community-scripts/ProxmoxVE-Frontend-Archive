# ProxmoxVE Frontend (Archive)

> **This repository is archived.** It contains the former frontend of [community-scripts/ProxmoxVE](https://github.com/community-scripts/ProxmoxVE), extracted with full git history.

The frontend was a Next.js application that powered [community-scripts.github.io/ProxmoxVE](https://community-scripts.github.io/ProxmoxVE/). It was removed from the main repository as part of a restructuring effort. The JSON data files that the frontend consumed remain in the main repo under `json/`.

## Tech Stack

- Next.js 15 / React 19 / TypeScript
- Tailwind CSS / shadcn/ui
- Bun as package manager
- Static export to GitHub Pages
